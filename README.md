# 📖 PostBook

A full-stack social media application built with **Node.js**, **Express.js**, **MongoDB**, **EJS**, and **Tailwind CSS**.

## ✨ Features

- 🔐 User Authentication (Signup & Login)
- 🔑 Password Hashing with bcrypt
- 🪪 JWT Authentication
- 📝 Create Posts
- ✏️ Edit Your Posts
- 🗑️ Delete Your Posts
- 👤 User Authorization
- 🎨 Responsive UI with Tailwind CSS

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Tailwind CSS
- JWT
- bcrypt

## 🚀 Installation

```bash
git clone https://github.com/<your-username>/postbook.git

cd postbook

npm install

npm start
```

## ⚙️ Environment Variables

Create a `.env` file in the root directory.

```env
PORT=3000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
